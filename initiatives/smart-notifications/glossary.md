# Smart Notifications Glossary

## Notification
A message delivered to a User through one or more Channels, triggered by a system event or scheduled action.

## Channel
A delivery mechanism for Notifications. Supported Channels include email, push, and SMS.

## User
A registered individual in the system who can receive Notifications and manage their Preferences.

## Preference
A User-defined setting that controls which Channels are enabled for each Notification type.

## Template
A predefined message layout used to render Notification content, supporting variable substitution.

## Digest
A bundled summary of multiple Notifications delivered as a single message at a scheduled interval.
